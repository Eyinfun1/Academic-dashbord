# Academic Advising Dashboard

A prototype advising tool to help academic advisors identify at-risk students and take timely action.

## Features
- Student risk dashboard (GPA, attendance, missing assignments)
- Step-by-step decision tree for intervention
- Action panel (messaging, scheduling, referrals)
- Supervisor role & delegation management
